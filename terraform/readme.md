# Production-grade EKS Platform (Reference)

This repository contains a reference implementation of a production-ready
Kubernetes platform on AWS, built with Terraform and Kubernetes best practices.

## Goals
- Reproducible infrastructure
- Clear separation of environments
- Infrastructure as Code
- Production-oriented design decisions

## Scope (initial)
- AWS VPC
- Amazon EKS
- Terraform modules
- CI/CD foundations
- Observability basics
